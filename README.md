# SkillSwap Full App (Starter)

This is a full-stack starter for SkillSwap built with Next.js (React).
It includes:
- Landing page with skill feed
- Profile page
- Skill detail page
- Simple mock API (Next.js API routes using in-memory JSON)

## How to run

1. Make sure you have **Node.js 18+** installed.
2. Open a terminal in this folder.
3. Run:

   npm install
   npm run dev

4. Open http://localhost:3000 in your browser.

You can deploy this to Vercel or any other Node/Next hosting.
